Your Prompt Guide
for Virtual Assistant Work

Ready-to-use prompts for ChatGPT (free plan). Copy, paste, and get results in seconds.

Paste email content directly into ChatGPT with these prompts. Works on the free plan — no inbox access needed. Copy the prompt, fill in the brackets, paste your email text at the bottom.

Sort & label
Organize a batch of emails
Here are emails from my client's inbox. For each one, suggest: (1) a short label/category, (2) whether to keep, archive, or delete, and (3) a priority level — high, medium, or low. Be concise. [paste emails here]
Draft a reply
Write a professional email response
Write a professional but friendly reply to this email. Keep it under 100 words. Tone: [warm / formal / direct]. My name is [your name]. [paste email here]
Label system
Build a label structure for a client
My client is a [type of business, e.g. real estate agent / coach / photographer]. Suggest a simple Gmail label system — no more than 8 labels — that would keep their inbox organized. Include sub-labels where useful.
Summarize
Get the key points from a long email thread
Summarize this email thread in 3–5 bullet points. Include: who needs to do what, any deadlines mentioned, and any unresolved questions. [paste thread here]

Use these to plan and clean up folder structures. No file access needed — just describe your situation and let ChatGPT suggest a system.

💬 Tip: List your current folders in the prompt. The more specific you are about what you actually have, the more useful the suggestions will be.
Folder structure
Design a clean folder system
I'm a virtual assistant managing work for multiple clients. Help me design a simple, logical Google Drive folder structure. I need to track: client deliverables, ongoing projects, invoices, and reference documents. Suggest a top-level structure and sub-folders. Keep it simple enough that I can find anything in under 30 seconds.
Fix confusion
Decide where a file belongs
I have these folders: [list your folders, e.g. *Clients, *2026, Projects, Archive]. I'm not sure where to put files like [describe the file]. Tell me the best folder for each, and suggest a rule I can follow going forward so I'm consistent.
Naming convention
Create consistent file naming rules
Help me create a file naming convention for my VA work. I have files like contracts, meeting notes, drafts, and final deliverables — often for multiple clients. I want names that are sortable by date and easy to search. Give me a template and 5 examples.

Use ChatGPT to write all the copy, then paste it into Canva (free at canva.com) to design. This combo beats either tool alone — AI for words, Canva for visuals.

Write copy
Generate flyer text for a client
Write flyer copy for [business name], a [type of business]. The flyer is for [event / promotion / service]. Include: a punchy headline (under 8 words), 2–3 short benefit statements, a clear call to action, and a contact info placeholder. Tone: [professional / friendly / urgent / fun].
Design direction
Get design advice before opening Canva
I'm designing a flyer in Canva for [describe the business and purpose]. Suggest: a color palette (2–3 colors), a layout style (e.g. bold headline top, image center), what type of image would work best, and any fonts that fit the vibe. Keep it beginner-friendly.
Revise copy
Improve existing flyer text
Here is the current text on a flyer I'm designing. Make it more compelling and concise. Cut anything that doesn't need to be there. Keep the tone [professional / casual / exciting]. [paste current flyer text]

Before building, use AI to plan and pitch. Then use Carrd or Framer (both have free tiers) to build — no coding needed. Always confirm interest before spending time building.

💬 Start with the outreach prompt below — pitch the idea before you build. This saves hours of work if the business says no.
Outreach first
Write a pitch to a local business before building
Write a short, friendly cold outreach message to a local [type of business, e.g. bakery / hair salon / plumber] that doesn't have a website. I want to offer to build them one. Don't be pushy. Focus on the benefit to them (more customers, look professional). Keep it under 100 words and include a question to start a conversation.
Plan the site
Create a simple website plan
I'm building a simple website for a local [type of business]. They have no website yet. Suggest: the 4–5 pages to include, what content goes on each page, and what info I need to collect from the business owner before I start. Keep it simple — this is a starter site, not a full e-commerce build.
Write content
Generate homepage copy
Write homepage copy for [business name], a [type of business] located in [city]. Include: a headline that grabs attention, a short "about" paragraph (3 sentences), a list of 3–4 services, and a call-to-action button label. Friendly and local in tone.

These habits will make every prompt work better — no matter the task or tool you're using.

1

Always give context about the business

The more context you give, the less editing you'll need to do.


Instead of: "Write an email."

Try: "Write an email for my client, a life coach who works with women in career transitions. Professional but warm tone."

2

Tell it the format you want

Adding format instructions saves a lot of back-and-forth. Try phrases like:

  • Give me 3 options
  • Use bullet points
  • Keep it under 100 words
  • Format as a table
  • Write it as if speaking to a small business owner
3

Ask it to improve your draft, not start from scratch

Write a rough version yourself and say: "Here's my draft. Make it more professional and fix any awkward phrasing. Keep my meaning, just improve the wording."


This saves time and keeps your voice in the final result.

4

Save prompts that work

When a prompt gives you a great result, save it in a Google Doc called "Prompts that work." Over time you'll build a personal library tailored to your clients — no starting from scratch each time.

5

When free ChatGPT hits its limit, switch tools

Free ChatGPT has a daily message limit. When you hit it, try:

  • Claude.ai (free plan) — great for writing and organization tasks
  • Google Gemini (free) — connects well with Google Drive and Gmail
  • Keep both bookmarked and alternate between them
